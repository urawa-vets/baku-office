# baku-office — 自社専用AIの相棒（クライアントアプリ）

**「自社専用AIの相棒」を、あなたの団体が丸ごと所有する** ための baku-office クライアントアプリです。
AIと対話しながら自社に必要な機能（アプリ）を作り・載せ替え・育て、それを **あなたのCloudflareアカウントに完全所有** します。
会計・庶務は、あらかじめ入っている機能の一例にすぎません。
このリポジトリは **配布用のビルド済みバンドル** です（ソースは非公開・当社管理）。

## ワンクリック導入（Deploy to Cloudflare）

[![Deploy to Cloudflare](https://deploy.workers.cloudflare.com/button)](https://deploy.workers.cloudflare.com/?url=https://github.com/baku-team/baku-office-app)

1. ボタンを開き、Cloudflareに**団体のCloudflareアカウント**でサインイン。
2. Cloudflareがこのリポジトリを**あなたのGitHubへ複製**し、**D1・KVをあなたのアカウントに自動作成**してデプロイ。
3. **秘密鍵などの設定は不要**：暗号鍵（MASTER_KEY）は初回に自動生成され、ライセンス検証鍵・ログインは当社ホストと自動連携します。
4. 申込時に届いた**アクティベーションURL**を開く → ライセンスを自動取得 → **「Googleでログイン」**（団体側のGoogle設定は不要）。

## 使う画面は4つ

- **ホーム**：お知らせ、API/DB/ストレージ使用量、導入したアプリのウィジェット（例：総会員数・当月取引数）。
- **AI**：相棒とのチャット（会話の保存・切替、モデル選択 Gemini/Claude/ローカル）。ここで「〜するアプリを作って」と頼めば、自社用アプリ作りも始まります。
- **アプリ**：機能（アプリ）の導入・削除、外部からの取り込み、AIと作った草案の公開申請。
- **設定**：連携（APIキー）・プラン課金・カスタマイズ・運用をカテゴリ別に集約。

## 導入後の設定（管理画面から）

- **連携設定**：AIを使う場合のみ Gemini／（Pro）LINE・Claude のAPIキーを登録（保存時に検証→暗号化保存）。AI不要なら設定不要で会計・庶務が使えます。
- **プラン・課金**：**Free（無料）**はそのまま利用可。**Plus（AI・アプリ開発）／Pro（エージェント）**はアップグレード。
- **ログイン**：組織＝Google（当社ホスト経由の中継。団体側の設定不要）。

## 更新

当社が新バージョンを公開すると、アプリの「**設定 → アプリの更新**」に通知が出ます。更新は**同じプロジェクトの再ビルド**で行われ、再ビルド時に署名検証済みの最新版が自動で取り込まれます（データ・設定は保持）。

- **手動更新（基本）**：Cloudflare の **Deployments** で「**Create deployment**」または「**Retry**」を1回押す（＝再ビルド）。数分で最新版になります。
- **自動更新（任意・初回のみ2〜3分）**：アプリの「設定 → アプリの更新」で **Deploy Hook** を登録しておくと、新版公開時に1タップ／自動で更新できます。

⚠️ **Deploy ボタンの押し直し（最初からやり直し）はしないでください**（D1・KV が切り離されデータが孤立します）。当社はあなたのアカウントに入りません。

### 更新が反映されないとき（古い版で導入した場合・一度きりの手当て）

ごく初期に導入したリポジトリは、更新の「取り込み部分」が古く、再ビルドしても最新版が入らないことがあります。その場合は本配布リポジトリ（`baku-team/baku-office-app`）の `prebuild-update.mjs` と `wrangler.jsonc` の**2ファイルの内容を、あなたのリポジトリの同名ファイルへ丸ごと貼り替えて commit** すると、以後は自動で最新を取り込みます。手順が不安な場合は**当社サポート**へご連絡ください（画面共有で伴走します）。
